# Modern Landing Page

A modern and responsive landing page built using HTML and CSS.

Designed with a clean user interface, responsive layout, and multiple sections including Features, Services, Pricing, and Contact.

## 🚀 Features

- Modern UI Design
- Fully Responsive Layout
- Hero Section
- Features Page
- Services Page
- Pricing Plans
- Contact Form
- Mobile Friendly Design
- Smooth Hover Effects

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
